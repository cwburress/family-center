A localized Python/Flask family center dashboard designed to run on a Raspberry Pi and display on a wall-mounted Lubuntu monitor in kiosk mode. It integrates with Google Calendar for daily scheduling and leverages the National Weather Service (NWS) API for pinpoint weather data. Features include a 5-day forecast, conditional radar loop triggering on precipitation or active weather polygons, severe alert audio sirens, and a custom logic matrix that recommends the kids' school outfits based on morning and afternoon temperature shifts.
