Family Center Dashboard Guide

Welcome to the Family Center display. Here is a quick breakdown of how to read the interface and access hidden details.

Feature Overview

  • Left Panel: Displays the live clock, today's school lunch/activity, and a clothing recommendation calculated from the day's high and low temperatures.
  • Middle Panel (Today): Highlights events happening exclusively today. All-day events are highlighted in light blue.
  • Right Panel (Schedule): A rolling view of upcoming calendar events for the next few days.
  • Bottom Panel (Weather): Shows the 5-day forecast. If rain or severe weather is in the area, the 5th day will drop off to make room for a live weather radar.

Touch Features

  • Event Details: Tap any event in the Today or Schedule columns to pull up a window with exact times, location data, and full descriptions.
  • Full Lunch Menu: The main screen only shows a preview. Tap the School Lunch box to open the complete list of entrees and sides.
  • Fullscreen Map: When the radar is active, tap anywhere on the map to expand it to fullscreen. Tap it again to return to the normal dashboard view.
  • Schedule Auto-Reset: If you scroll down through the Future Schedule, you don't need to scroll back up. It will automatically snap back to the top after 15 seconds of inactivity.

The "Fire Drill" Radar

The NEXRAD radar automatically appears based on live NWS precipitation sweeps and active weather alerts. If you want to manually force the radar to open on a clear day (for a drill or to check regional storms), tap the main clock 5 times rapidly. The clock text will turn red, and the radar will deploy.

Admin Setup: JSON Configuration

The dashboard polling threads check local JSON files for school data. To keep the UI populated, update these files ensuring strict JSON syntax.

menu.json

Top-level keys must be formatted as YYYY-MM-DD strings. The app.py script matches this exactly against the system's current local date. Both entrees and sides must be arrays, even if empty.

{
  "2026-05-01": {
    "entrees": ["Cheese Filled Breadsticks", "Ham & Cheese Sub"],
    "sides": ["Sunshine Carrots", "Veggie Medley"]
  },
  "2026-05-02": {
    "entrees": [],
    "sides": []
  }
}

activity.json

Top-level keys are exact string matches for the full day of the week (e.g., "Monday"). The nested dictionary maps the individual display name to their assigned activity.

{
  "Monday": {
    "Child01": "Art",
    "Child02": "Library"
  },
  "Tuesday": {
    "Child01": "Computer Lab",
    "Child02": "Art"
  }
}
